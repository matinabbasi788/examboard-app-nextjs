# راهنمای شروع سریع

این راهنما برای شروع سریع پروژه طراحی شده است. برای اطلاعات کامل، به [DOCUMENTATION.md](./DOCUMENTATION.md) مراجعه کنید.

## نصب سریع

```bash
# 1. نصب dependencies
pnpm install

# 2. ایجاد فایل .env.local
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local

# 3. اجرای development server
pnpm dev
```

برنامه روی `http://localhost:3000` اجرا می‌شود.

## استقرار سریع با Docker

```bash
# 1. ایجاد فایل .env
echo "NEXT_PUBLIC_API_URL=http://your-backend-api-url:8000" > .env

# 2. ساخت و اجرا
docker-compose up -d

# 3. بررسی وضعیت
docker-compose ps
```

## استفاده سریع

### 1. لاگین

1. به `http://localhost:3000` بروید
2. روی "ورود به سیستم" کلیک کنید
3. username و password را وارد کنید

### 2. ایجاد ترم

1. به `/terms` بروید
2. فرم را پر کنید و "افزودن ترم" را بزنید

### 3. ایجاد مکان

1. به `/rooms` بروید
2. فرم را پر کنید و "افزودن مکان" را بزنید

### 4. ثبت امتحان

1. به `/exams` بروید
2. یک ترم را انتخاب کنید
3. فرم را پر کنید و "ثبت امتحان" را بزنید

### 5. وارد کردن از Excel

1. به `/exams/import-export` بروید
2. یک ترم را انتخاب کنید
3. فایل Excel را انتخاب کنید
4. روی "وارد کردن" کلیک کنید

## فرمت Excel

حداقل ستون‌های مورد نیاز:
- **عنوان** یا **نام درس**: نام امتحان
- **کد درس**: کد درس (اختیاری)

ستون‌های اضافی:
- **زمان امتحان**: `1404/10/18 از 10:30 تا 12:00`
- **تاریخ (میلادی)**: `2025-01-07`
- **تاریخ (شمسی)**: `1404/10/18`
- **ساعت**: `10:30`
- **مدت (دقیقه)**: `90`
- **تعداد دانشجویان**: `100`
- **محل برگزاری**: `سالن ۱۰۱`

## نکات مهم

1. **Backend API**: باید در حال اجرا باشد
2. **CORS**: باید در backend تنظیم شود
3. **Token**: بعد از لاگین، token در cookie ذخیره می‌شود

## مشکلات رایج

### API کار نمی‌کند
- بررسی کنید که backend در حال اجرا است
- `NEXT_PUBLIC_API_URL` را بررسی کنید

### تاریخ نمایش داده نمی‌شود
- بررسی کنید که allocation ایجاد شده است
- در صفحه `/exams` روی آیکون ✏️ کلیک کنید و allocation را ایجاد کنید

### Excel import کار نمی‌کند
- فرمت فایل را بررسی کنید
- ستون‌های الزامی را بررسی کنید
- Console browser را برای خطاها بررسی کنید
